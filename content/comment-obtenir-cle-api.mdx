# Comment obtenir votre clé d'API

Bienvenue ! Ce guide vous explique étape par étape comment obtenir votre clé d'API.

## Obtention de votre clé d'API

<Steps>

### 1. Créez votre compte sur Ikalas

Rendez-vous sur [https://ikalas.com](https://ikalas.com) et créez votre compte utilisateur.

### 2. Accédez au dashboard

Une fois connecté, allez sur [https://dashboard.ikalas.com](https://dashboard.ikalas.com)

### 3. Naviguez vers la section API

Dans le dashboard, cliquez sur l'onglet **"API"** ou allez directement sur [https://dashboard.ikalas.com/\_api](https://dashboard.ikalas.com/_api)

### 4. Créez votre clé d'API

Dans la section API, vous verrez un bouton **"Nouveau"**. Cliquez dessus pour générer votre clé d'API.

### 5. Copiez et sauvegardez votre clé

Votre clé d'API s'affichera sous un format similaire à : `IK-_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

⚠️ **Important** : Sauvegardez cette clé dans un endroit sûr. Elle ne sera plus visible après avoir quitté la page.

</Steps>
