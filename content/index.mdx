# Pour commencer

Bienvenue sur la documentation officielle de l'API d'Ikalas en français.
Vous trouverez ici toutes les informations nécessaires pour utiliser l'API d'Ikalas.

Ikalas propose une API unifiée qui vous donne accès à des centaines de modèles d'IA via un seul point d'entrée.
Voici comment commencer.

1. Créez votre compte sur Ikalas
2. Créez votre clé d'API
3. Prenez connaissance de cette documentation pour savoir comment utiliser l'API
4. Commencez à utiliser l'API

un exemple simple pour commencer

````
curl -d '{"system": "You are a useful assistant","prompt": "","models": "meta-llama/llama-3.3-70b-instruct:free","response_format": "default"}'\
  -X POST "https://ikalas.com/api/v1/api-llm-ia"\
  -H "accept: application/json"\
  -H "apikey: EICZM-DNP0-7NSN-YFQF"\
  -H "content-type: application/json"
```
````
