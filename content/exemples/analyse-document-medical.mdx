## Vue d'ensemble

L'API d'analyse de documents médicaux d'Ikalas permet aux professionnels de santé et aux individus de comprendre des documents médicaux complexes. Cette fonctionnalité analyse les rapports médicaux, études de recherche ou dossiers de santé téléchargés et fournit des insights concis et compréhensibles ainsi que des recommandations liées à la santé.

## Fonctionnalités principales

- **Analyse de documents** : PDF, PowerPoint, Word, TXT
- **Insights médicaux** : Résumé des points médicaux clés
- **Évaluation des risques** : Identification des risques de santé potentiels
- **Recommandations pratiques** : Conseils basés sur les informations fournies
- **Points d'attention** : Éléments nécessitant un suivi médical spécial

## Endpoint API

```
POST https://ikalas.com/api/v1/analyse-documents-medicaux-ia
```





## Paramètres de la requête

| Paramètre | Type | Requis | Description |
|-----------|------|--------|-------------|
| `inputFiles` | File | Oui | Document médical à analyser (PDF, PPT, PPTX, DOC, DOCX) |
| `prompt` | String | Non | Question spécifique ou instruction d'analyse |

## Exemple de prompt
```
Veuillez analyser ce document médical et fournir : 
1. Un résumé des points médicaux clés ; 
2. Une évaluation des risques potentiels pour la santé ; 
3. Des recommandations pratiques basées sur les informations fournies ; 
4. Les points nécessitant une attention particulière ou un suivi médical. Présentez les informations de manière structurée et professionnelle, en utilisant la terminologie médicale appropriée tout en restant accessible.
```

### Requête JavaScript


  ```javascript
    const axios = require('axios');
    const FormData = require('form-data');
    const fs = require('fs');

    let data = new FormData();
    data.append('file', fs.createReadStream('/Users/ikalas/samples/test-medical-report.pdf'));
    data.append('prompt', 'Analyser ce rapport médical et fournissez un résumé des points clés et des recommandations');

    let config = {
      method: 'post',
      maxBodyLength: Infinity,
      url: 'https://ikalas.com/api/v1/analyse-documents-medicaux-ia',
      headers: {
        'accept': 'application/json',
        'apikey': 'VOTRE_CLE_API_ICI',
        ...data.getHeaders()
      },
      data : data
    };

    axios.request(config)
    .then((response) => {
      console.log(JSON.stringify(response.data));
    })
    .catch((error) => {
      console.log(error);
    });
    ```
