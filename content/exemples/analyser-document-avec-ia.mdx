# Exemple : Analyse de documents avec l'IA

Cet exemple vous montre comment utiliser l'API d'analyse de documents d'Ikalas pour extraire et analyser le contenu de vos documents PDF.

## Fonctionnalités

- **Analyse de documents PDF** : Extraction et résumé automatique
- **Prompts personnalisables** : Posez des questions spécifiques sur vos documents
- **Support multilingue** : Analyse en français, anglais et autres langues
- **Traitement sécurisé** : Vos documents sont traités de manière sécurisée

## Prérequis

- Une clé API Ikalas valide
- Node.js installé sur votre machine
- Un document PDF à analyser

## Installation des dépendances

```bash
npm install axios form-data fs
```

## Exemple de code

<Tabs items={['JavaScript', 'cURL']}>
  <Tabs.Tab>
    ```javascript
    const axios = require('axios');
    const FormData = require('form-data');
    const fs = require('fs');

    let data = new FormData();
    data.append('file', fs.createReadStream('/Users/ikalas/samples/test-medical-report.pdf'));
    data.append('prompt', 'resume');

    let config = {
      method: 'post',
      maxBodyLength: Infinity,
      url: 'https://ikalas.com/api/v1/ask-document',
      headers: {
        'accept': 'application/json',
        'apikey': 'VOTRE_CLE_API_ICI',
        ...data.getHeaders()
      },
      data : data
    };

    axios.request(config)
    .then((response) => {
      console.log(JSON.stringify(response.data));
    })
    .catch((error) => {
      console.log(error);
    });
    ```

  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    curl -X POST "https://ikalas.com/api/v1/ask-document" \
      -H "accept: application/json" \
      -H "apikey: VOTRE_CLE_API_ICI" \
      -F "file=@/Users/ikalas/samples/test-medical-report.pdf" \
      -F "prompt=resume"
    ```
  </Tabs.Tab>
</Tabs>

## Paramètres de l'API

| Paramètre | Type   | Description                               | Obligatoire |
| --------- | ------ | ----------------------------------------- | ----------- |
| `file`    | File   | Le document PDF à analyser                | ✅          |
| `prompt`  | String | La question ou instruction pour l'analyse | ✅          |
| `apikey`  | String | Votre clé API Ikalas                      | ✅          |

## Exemples de prompts

Voici quelques exemples de prompts que vous pouvez utiliser :

- `"resume"` - Génère un résumé du document
- `"Quels sont les points clés ?"` - Extrait les points importants
- `"Analyse les risques mentionnés"` - Identifie les risques dans le document
- `"Traduis en anglais"` - Traduit le contenu
- `"Extrais les dates importantes"` - Identifie les dates clés

## Gestion des erreurs

Voici les étapes pour résoudre les problèmes courants :

### 1. Vérifiez votre clé API

Assurez-vous que votre clé API est valide et active dans votre compte Ikalas.
Vous pouvez vérifier votre clé d'api dans le [dashboard](https://dashboard.ikalas.com)

### 2. Vérifiez la taille du fichier

Les tailles acceptées dépendent de votre abonnement. Voir les détails dans la page [pricing](https://ikalas.com/app/pricing)
