# Pense bête

Quelques informations utilises à savoir pour s'y retrouver dans l'API.

| Point de terminaison API               | Description                     |
| -------------------------------------- | ------------------------------- |
| `https://ikalas.com/api/v1`            | Url de base pour les appels API |
| `https://ikalas.com/api/v1/api-llm-ia` | Url pour les appels API IA      |

| Urls                                | Description                            |
| ----------------------------------- | -------------------------------------- |
| `https://dashboard.ikalas.com/_api` | Pour créer et retrouver vos clés d'API |

## Authentification

Incluez votre clé API dans toutes les requêtes. Votre clé d'API est ici : [https://dashboard.ikalas.com/\_api](https://dashboard.ikalas.com/_api)

## Formats de requête et de réponse

L'API Ikalas utilise JSON pour toutes les requêtes et réponses. Assurez-vous d'inclure l'en-tête `Content-Type: application/json` dans vos requêtes.

Exemple de requête :

```json
{
  "system": "You are a useful assistant",
  "prompt": "Date de naissance Napoléon",
  "models": "meta-llama/llama-3.3-70b-instruct:free",
  "response_format": "default"
}
```
